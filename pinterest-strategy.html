Strategy: Create 5-10 pins per day across 6 boards. Focus on keyword-rich titles and descriptions. Link every pin directly to the quiz page. Results typically compound from month 2 onward — this is a long-shelf-life engine, not a quick-spike channel.

Pinterest SEO & Growth Tips

Keyword Strategy

Pinterest is a search engine. Every pin title and description must contain the primary keyword in the first 40 characters. Use long-tail keywords that match intent: "free astrology reading online" not just "astrology."

  • Use Pinterest's search bar autocomplete to find exact keyword phrases
  • Include keywords in both pin title AND description
  • Board names must also be keyword-rich

Image Specifications

Pinterest favors vertical images. Use 1000x1500px (2:3 ratio) for standard pins or 1000x2100px for tall pins. Dark, high-contrast designs with clear text overlay perform best in this niche.

  • Text must be readable at thumbnail size
  • Include the COSMO logo on every pin
  • Gold on dark performs well against the white Pinterest background

Fresh Pin Strategy

Pinterest rewards fresh content — never repin the same image twice. Create 3-5 different images for the same URL and pin them to different boards over time. This multiplies impressions without duplicating content.

  • Same URL, different image = fresh pin
  • Spread pins across boards for same content
  • Schedule with Tailwind for consistent cadence

Conversion Optimization

Always link directly to the quiz (index.html), not the homepage. The quiz entry point converts 3x better than a sales page for Pinterest traffic. Add UTM parameters to track Pinterest conversions in GA4.

  • UTM: ?utm_source=pinterest&utm_medium=organic
  • Set up Pinterest Tag for conversion tracking
  • A/B test pin images monthly — iterate fast
